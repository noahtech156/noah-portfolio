# Noah's Portfolio

A dynamic portfolio website built with Node.js, Express, and EJS.

## Features

- Dynamic content management via admin panel
- Responsive design with Tailwind CSS
- DevOps integration with CI/CD

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm start` for production or `npm run dev` for development
4. Visit `http://localhost:3000`
5. Admin panel at `http://localhost:3000/admin` (username: admin, password: password)

## Deployment

This project is set up for deployment on Vercel with GitHub Actions.

1. Create a Vercel account and project
2. Add VERCEL_TOKEN, VERCEL_ORG_ID, VERCEL_PROJECT_ID to GitHub secrets
3. Push to main branch to trigger deployment

## DevOps

- Version control with Git
- CI/CD with GitHub Actions
- Hosting on Vercel
